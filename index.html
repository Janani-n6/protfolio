Spring Boot CRUD Application

In this project, I developed a CRUD application using Spring Boot, which demonstrates the fundamental operations of Create, Read, Update, and Delete. The application was designed to manage a simple database of user information. I implemented RESTful web services to handle HTTP requests and responses, ensuring a robust and scalable backend. This project helped me gain practical experience with Spring Boot, JPA (Java Persistence API) for database interactions, and the creation of RESTful APIs. Additionally, I used Postman to verify the correctness of the API endpoints.
